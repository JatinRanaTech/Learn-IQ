🎓 LearnIQ Kids

LearnIQ Kids is an interactive educational website that helps children learn while having fun. The platform includes brain-training games, quizzes, learning activities, colorful animations, and a user-friendly interface designed specifically for young learners.

✨ Features
• Interactive educational games
• Brain training and memory challenges
• Child-friendly colorful UI
• Responsive design for mobile and desktop
• Engaging learning experience
• Fun and educational activities

🛠️ Technologies Used
• HTML5
• CSS3
• JavaScript

The goal of LearnIQ Kids is to make learning enjoyable, interactive, and accessible for children of all ages.
